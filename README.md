# 안가은 Portfolio

Apple Developer, Linear, Vercel 감성의 미니멀 GitHub Pages용 정적 포트폴리오입니다.

## 실행

VSCode Live Server로 `index.html`을 열거나 아래 명령어를 실행하세요.

```bash
python -m http.server 8000
```

브라우저에서 접속합니다.

```text
http://localhost:8000
```

## 썸네일 교체

`assets/images/` 폴더에 실제 프로젝트 캡처 이미지를 아래 파일명으로 넣으면 자동 반영됩니다.

```text
youtube-growth.png
musinsa-recommendation.png
hrd-radar.png
education-guide-generator.png
```

## 수정 포인트

- GitHub 링크: `index.html`에서 `https://github.com/` 검색 후 본인 주소로 변경
- Email: `example@email.com` 검색 후 본인 이메일로 변경
- Demo 링크: `href="#"`를 실제 배포 URL로 변경
- 프로필 이미지: `assets/avatar.png` 추가
